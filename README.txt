# The Art of Big Social Data/MizzouSENG

This project uses an API from the Huffington Post that provides polling information for primaries, elections, and various topics. It focuses on the Republican(GOP) candidates for the 2016 Presidential Election.

[jQuery](https://github.com/The-Art-of-Big-Social-Data/MizzouSENG/blob/master/jquery-1.12.0.min.js)
[D3.js](https://github.com/The-Art-of-Big-Social-Data/MizzouSENG/blob/master/d3.v3.min.js)
[Huffing Post API](http://elections.huffingtonpost.com/pollster/api)
